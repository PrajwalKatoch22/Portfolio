BAD BOT

Unreal Engine 5
Blueprint Graph Watch Gameplay

Project Overview

Bad Bot is a fast-paced aerial combat prototype built in Unreal Engine 5. The project explores high-mobility gameplay through a drone-style player controller with full six-degrees-of-freedom movement. The core idea was to create fluid aerial navigation combined with responsive combat mechanics, allowing players to maneuver through a dense environment while engaging AI-controlled enemies. The gameplay focuses on fast reaction times, spatial awareness, and strategic positioning in three-dimensional space.

My Role

  • Designed and implemented the 6-DOF drone movement system.
  • Developed the laser projectile combat mechanics.
  • Implemented AI combat behavior for enemy sentry drones.
  • Handled gameplay scripting using Blueprint in Unreal Engine 5.
  • Integrated and optimized the environment for aerial navigation.
  • Balanced player movement, combat responsiveness, and enemy difficulty.

Key Technical Features

  • 6-DOF Flight Mechanics
    • Developed a drone-style player controller supporting six degrees of freedom.
    • Implemented smooth acceleration and deceleration curves to simulate propulsion-based movement.
    • Created responsive aerial control for navigating tight spaces and vertical environments.
  • Dynamic Projectile System
    • Implemented a high-frequency laser weapon system.
    • Used object pooling techniques for efficient projectile spawning and performance optimization.
    • Added precise collision detection using line traces or sphere collision triggers.
    • Integrated damage events and visual hit feedback.
  • AI Combat Behavior
    • Designed reactive enemy sentry bots using a simple AI state machine.
    • Implemented distance-maintaining behavior to keep enemies in combat range.
    • Created predictive firing patterns to challenge player movement.
    • Triggered visual and audio feedback upon enemy destruction.
  • Level & Environment Integration
    • Integrated a detailed Asian Village environment for aerial gameplay.
    • Focused on vertical level design allowing players to fly over rooftops and through gates.
    • Ensured smooth navigation through both exterior and interior spaces.
    • Optimized environment assets to maintain gameplay performance.

Technologies Used

  • Unreal Engine 5
  • Blueprint Visual Scripting
  • Gameplay Framework
  • AI Behavior Systems
  • Line Trace Collision Detection
  • Particle Effects / VFX

Key Learning Outcomes

  • Designing fluid aerial movement systems using 6-DOF mechanics.
  • Implementing efficient projectile systems using object pooling.
  • Developing reactive AI combat behaviors.
  • Balancing high-speed movement with environmental level design.
  • Optimizing gameplay performance in complex environments.

JetPack simulation

Unreal Engine 5 C++
Blueprint Graph Watch Gameplay

Project overview: Developed a 3D platforming prototype focusing on verticality and resource management. The project features a custom character controller integrated with a jetpack mechanic, requiring players to balance movement and fuel consumption to navigate a series of floating environmental puzzles.

My Role

  • Designed and implemented the jetpack-based movement system.
  • Developed the core gameplay mechanics using Unreal Engine Blueprint.
  • Created the resource management system controlling jetpack fuel.
  • Designed and scripted interactive environmental elements.
  • Implemented HUD elements to visually communicate fuel levels to the player.
  • Built the prototype level to test vertical navigation and player mobility.

Key technical features

  • Advanced character controller
    • Developed a custom jetpack movement system that adds upward impulse and modifies gravity scales for a “floaty” yet responsive feel.
    • Implemented a seamless transition between standard locomotion and aerial jetpack states.
  • Dynamic resource & fuel system
    • Integrated a real‑time fuel‑depletion logic during jetpack use.
    • Created an item‑collection system (yellow fuel orbs) that triggers instant refuelling and provides visual feedback through UI/particle effects.
  • Environmental scripting & level design
    • Interactive platform triggers – engineered pressure‑plate / button mechanics that use Blueprint communication to move platforms and shift the environment.
    • Modular level layout – designed a series of floating platforms using a modular kit to test the limits of the player’s aerial mobility.
  • User‑interface integration
    • Functional HUD with a fuel bar that updates dynamically from the character’s internal variables.
    • Win‑state screen with a Play Again / Quit menu upon level completion.
